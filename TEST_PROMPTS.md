# 结构化渲染测试提示词

## 折线图 / 柱状图（chart）

```
帮我列一下过去5年（2020-2024年）中国GDP增速数据，用图表展示
```

```
用柱状图展示一下各大洲面积对比
```

```
用折线图展示比较 Python、JavaScript、Java、Go、Rust 近五年的 TIOBE 指数趋势（可用示例数据）
```

---

## 表格（datatable）

```
列出常见编程语言（Python、Java、Go、Rust、TypeScript）的主要特点对比，包括性能、学习难度、适用场景
```

```
给我一个A股主要指数（上证、深证、创业板、科创板）的基本信息对比表
```

```
列出主流云厂商（阿里云、腾讯云、AWS、Azure、GCP）在计算、存储、数据库方面的产品名称对比表
```

---

## Mermaid 流程图 / 架构图

```
用流程图画出用户注册的完整流程，包括填写信息、邮箱验证、设置密码
```

```
画一个电商订单处理的流程图，从下单到送达
```

```
用 Mermaid 画出微服务架构图，包含网关、用户服务、订单服务、支付服务
```

```
用 Mermaid 画出 Git Flow 的分支模型
```

---

## 混合输出（图表 + 文字说明）

```
分析一周天气数据（北京），用折线图展示气温变化，并附上文字说明
```

```
给我一份 Redis 与 Memcached 的对比，先用文字总结，再用表格列出详细参数差异
```

---

## 提示技巧

如果 AI 没有自动输出可视化块，可以在问题末尾加：
- `请用 echart 图表展示`
- `请用表格整理`
- `请用 mermaid 流程图`
